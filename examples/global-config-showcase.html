Global Configuration System

All components now use a unified global configuration system that provides:

  • Consistent Sizing: Unified size system across all components (xs, sm, md, lg, xl)
  • Theme Integration: All colors, spacing, and design tokens from global config
  • TailwindCSS Classes: Dynamic class generation from configuration
  • Label Positioning: Configurable label layouts (top, left, over, floating)
  • State Management: Consistent hover, focus, active, and disabled states
  • Validation System: Unified validation with proper error handling
